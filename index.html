О себе

Начинающий WEB-разработчик. Окончил университет КазНАУ (г.Алматы). Знаю английский на среднем уровне. Верстаю адаптивные сайты в точности по макету. Постоянно развиваюсь, совершенствую свои навыки, изучаю что-то новое для себя.

HTML

Изучил язык гипертекстовой разметки текста — HTML. Научился делать разметку веб-страницы, применять теги и атрибуты на сайте.

CSS

Изучил таблицу каскадных стилей CSS. Научился стилизовать элементы на сайте с помощью селекторов, атрибутов и классов тегов.

JavaScript

Изучил язык программирования — JavaScript для создания интерактива на сайте.

Верстка

Изучил верстку страниц сайта по PSD макету. Научился верстать адаптивно и кроссбраузерно с применением медиа запросов.

Bootstrap

Изучил фреймворк CSS — Bootstrap. Научился применять готовую сетку Bootstrap в своих проектах.

Git

Изучил систему констроля версии Git для работы с удаленными репозиторями. Научился пушить, пулить, делать коммиты, создавать ветки и т.д.
